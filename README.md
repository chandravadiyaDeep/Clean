# Clean

A simple open-source command-line tool to clean messy CSV files in seconds.

## Why Clean?

* Eliminate repetitive data-cleaning tasks.
* Save time before data analysis or machine learning.
* Use one command instead of writing repetitive Python scripts.
* Provide a simple tool that anyone can use.

## Why not Excel?

* Manual cleaning is slow.
* Difficult to reproduce the same steps.
* Not suitable for large datasets.
* Easy to make human errors.

## Why not Pandas?

* Requires Python programming knowledge.
* Repetitive cleaning code for common tasks.
* Beginners often struggle with data preprocessing.
* Clean aims to provide a simple command-line interface.

## Who is it for?

* Students
* Data Analysts
* Data Scientists
* ML Engineers
* Researchers
* Businesses
* Anyone working with CSV files

## Vision

Build the easiest and fastest open-source CSV cleaning tool.

The long-term goal is to let users clean datasets with a single command:

```bash
clean sales.csv
```

instead of writing dozens of lines of Python code.

## Planned Features

* Remove duplicate rows
* Handle missing values
* Clean column names
* Remove empty rows and columns
* Detect incorrect data types
* Generate cleaning reports
* Validate datasets
* AI-powered cleaning suggestions
